# Sample Projects

This folder contains demo-ready legacy sample workspaces for the hackathon.

Use these samples to validate:
- single-stack detection
- outdated framework and runtime discovery
- database/runtime combinations
- cross-platform mixed-repository analysis

Included samples:
- `java-spring-oracle-legacy`
- `angular-mssql-legacy`
- `react-node-postgres-legacy`
- `dotnet-sqlserver-legacy`
- `python-django-postgres-legacy`
- `mixed-enterprise-workspace`

Each sample is intentionally small, but it includes enough manifests, config files, and source code to make the scan results believable in a live demo.
